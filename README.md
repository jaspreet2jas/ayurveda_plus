# Ayurveda Plus 🌿

A modern full-stack Ayurvedic clinic management platform designed to simplify appointment booking, patient management, doctor consultations, and administrative operations.

## 📖 Overview

Ayurveda Plus is a comprehensive healthcare web application that bridges the gap between traditional Ayurvedic healing and modern digital healthcare systems. The platform allows patients to book appointments, doctors to manage consultations, and administrators to oversee clinic operations efficiently.

## ✨ Features

### 🏠 Home Page
- Modern and responsive landing page
- Ayurvedic clinic branding and wellness-focused design
- Featured treatments and services
- Appointment booking call-to-action
- Clinic statistics and ratings

### 📅 Appointment Booking
- Doctor selection based on specialization
- Treatment selection
- Consultation mode selection (Online / In-Clinic)
- Real-time appointment summary
- Appointment confirmation system

### 👤 Patient Portal
- Personalized patient dashboard
- Upcoming appointments tracking
- Medical history management
- Prescription access
- Health reports and wellness monitoring
- Profile management

### 🩺 Doctor Dashboard
- Daily appointment schedule
- Patient management
- Consultation notes
- Prescription management
- Performance analytics
- Appointment status tracking

### 🔐 Admin Panel
- Manage doctors and patients
- Appointment management
- Clinic operations monitoring
- User role management
- System analytics
